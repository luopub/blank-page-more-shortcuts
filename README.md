# 空白页快捷方式 Chrome 扩展

一个功能丰富的Chrome浏览器扩展，用于在新标签页显示最近访问网站的快捷方式，支持自定义显示格式和数量。

## 功能特性

- 🎯 **智能快捷方式**: 自动获取最近访问的网站，按域名去重
- ⚙️ **灵活设置**: 支持自定义显示数量（4-12个）和格式（网格/列表/卡片）
- 🎨 **美观界面**: 现代化设计，支持响应式布局
- 💾 **设置持久化**: 设置自动保存，下次打开时保持配置
- 🔧 **双重控制**: 支持页面内设置和扩展弹窗设置
- 🌐 **网站图标**: 可选择显示网站favicon图标

## 安装方法

1. 下载或克隆此项目到本地
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 扩展安装完成

## 使用说明

### 基本使用
- 安装后，打开新标签页即可看到快捷方式
- 点击快捷方式可直接访问对应网站

### 设置快捷方式
1. **页面内设置**: 点击右上角的⚙️按钮
2. **扩展弹窗设置**: 点击浏览器工具栏中的扩展图标

### 可配置选项
- **显示数量**: 4个、6个、8个、10个、12个
- **显示格式**: 
  - 网格布局：整齐的网格排列
  - 列表布局：垂直列表显示
  - 卡片布局：大卡片样式
- **网站图标**: 开启/关闭favicon显示

## 技术实现

### 核心技术
- **Manifest V3**: 使用最新的Chrome扩展API
- **Chrome Storage API**: 持久化存储用户设置
- **Chrome History API**: 获取浏览器历史记录
- **现代JavaScript**: ES6+语法，模块化设计

### 主要功能模块
1. **设置管理**: 加载、保存、同步用户配置
2. **历史记录处理**: 获取、过滤、去重最近访问的网站
3. **UI渲染**: 根据设置动态生成快捷方式界面
4. **事件处理**: 用户交互和设置更新

## 权限说明

扩展需要以下权限：
- `storage`: 保存用户设置
- `tabs`: 访问标签页信息
- `history`: 读取浏览器历史记录

## 开发说明

### 本地开发
1. 修改代码后，在 `chrome://extensions/` 页面点击刷新按钮
2. 打开新标签页查看效果

### 自定义样式
- 修改 `styles/newtab.css` 调整页面样式
- 修改 `styles/popup.css` 调整弹窗样式

### 功能扩展
- 在 `scripts/` 目录下添加新的JavaScript模块
- 修改 `manifest.json` 添加必要的权限

## 注意事项

1. 扩展只能访问普通网页的历史记录，无法访问Chrome内部页面
2. 网站图标通过Google Favicon服务获取，可能存在网络延迟
3. 历史记录获取限制为最近7天，最多50条记录
