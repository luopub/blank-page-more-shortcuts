# 빈 페이지 바로가기 Chrome 확장

사용자 지정 표시 형식과 수량을 지원하는 새 탭 페이지에 최근 방문한 웹사이트의 바로가기를 표시하는 기능이 풍부한 Chrome 브라우저 확장 프로그램입니다.

## 기능

- 🎯 **스마트 바로가기**: 도메인별로 중복 제거된 최근 방문한 웹사이트 자동 가져오기
- ⚙️ **유연한 설정**: 표시 수(10-50) 및 형식(그리드/목록/카드) 사용자 지정 지원
- 🔍 **즉시 검색**: 바로가기를 빠르게 필터링하고 찾기 위한 내장 검색 상자
- 📜 **기록 보기**: 도메인을 클릭하여 해당 웹사이트에서 방문한 모든 페이지 표시
- 🎨 **아름다운 UI**: 반응형 레이아웃이 있는 현대적 디자인
- 💾 **영구 설정**: 설정이 자동으로 저장되고 다음 열 때 복원됩니다
- 🔧 **이중 제어**: 페이지 내 설정 및 확장 프로그램 팝업 설정 모두 지원
- 🌐 **웹사이트 아이콘**: 지능적인 파비콘 로딩, 내부 네트워크 IP 주소 지원
- 🌍 **다국어 지원**: 중국어, 영어, 독일어, 프랑스어, 스페인어, 일본어, 한국어 지원
- 📊 **페이지 수**: 각 도메인의 기록 페이지 수 표시

## 설치

1. 이 프로젝트를 로컬 머신에 다운로드하거나 복제합니다
2. Chrome 브라우저를 열고 `chrome://extensions/`로 이동합니다
3. "개발자 모드"를 활성화합니다
4. "압축 해제된 확장 프로그램 로드"를 클릭합니다
5. 프로젝트 폴더를 선택합니다
6. 확장 프로그램 설치 완료

## 사용 가이드

### 기본 사용법
- 설치 후 새 탭을 열면 바로가기가 표시됩니다
- 모든 바로가기를 클릭하여 해당 웹사이트에 직접 액세스할 수 있습니다

### 설정
1. **페이지 내 설정**: 오른쪽 상단의 ⚙️ 버튼을 클릭합니다
2. **확장 프로그램 팝업 설정**: 브라우저 도구 모음의 확장 프로그램 아이콘을 클릭합니다

### 검색 기능
- 검색 상자에 키워드를 입력하여 바로가기를 실시간으로 필터링
- 웹사이트 제목과 URL 검색 모두 지원
- Enter 키를 누르거나 단순히 입력을 시작하여 검색을 트리거

### 기록 보기
- 여러 기록 페이지가 있는 도메인을 클릭하여 기록 모달을 엽니다
- 모달에는 해당 도메인에서 방문한 모든 페이지(최대 30개)가 표시됩니다
- 페이지 제목, URL 경로 및 방문 시간이 표시됩니다
- ESC 키 또는 외부를 클릭하여 모달을 닫을 수 있습니다

### 구성 가능한 옵션
- **표시 수**: 10, 20, 30, 40 또는 50개 항목
- **표시 형식**:
  - 그리드 레이아웃: 깔끔한 그리드 배열
  - 목록 레이아웃: 수직 목록 표시
  - 카드 레이아웃: 큰 카드 스타일
- **웹사이트 아이콘**: 파비콘 표시 켜기/끄기

## 기술 구현

### 핵심 기술
- **Manifest V3**: 최신 Chrome 확장 프로그램 API 사용
- **Chrome Storage API**: 사용자 설정을 위한 영구 저장
- **Chrome History API**: 브라우저 기록 액세스
- **Chrome i18n API**: 다국어 지원
- **모던 JavaScript**: ES6+ 구문, 모듈식 설계
- **Fetch API**: 내부 네트워크 IP의 아이콘 로딩 지원

### 주요 기능 모듈
1. **설정 관리**: 사용자 구성 로드, 저장, 동기화
2. **기록 처리**: 최근 방문한 웹사이트 가져오기, 필터링, 중복 제거
3. **도메인 기록 관리**: 각 도메인의 기록 페이지 목록 저장(최대 30개)
4. **검색 기능**: 바로가기의 실시간 필터링 및 검색
5. **모달 시스템**: 도메인 아래 기록 페이지 표시
6. **아이콘 로딩 시스템**: 다단계 폴백 메커니즘, 내부 IP 및 여러 아이콘 소스 지원
7. **UI 렌더링**: 설정에 따라 바로가기 인터페이스 동적 생성
8. **이벤트 처리**: 사용자 상호 작용 및 설정 업데이트

## 권한

확장 프로그램에는 다음 권한이 필요합니다:
- `storage`: 사용자 설정 저장
- `tabs`: 탭 정보 액세스
- `history`: 브라우저 기록 읽기

## 개발

### 로컬 개발
1. 코드를 변경한 후 `chrome://extensions/` 페이지에서 새로 고침 버튼을 클릭합니다
2. 새 탭을 열어 변경 사항을 확인합니다

### 스타일 사용자 지정
- `styles/newtab.css`를 수정하여 페이지 스타일을 조정합니다
- `styles/popup.css`를 수정하여 팝업 스타일을 조정합니다

### 기능 확장
- `scripts/` 디렉토리에 새 JavaScript 모듈 추가
- `manifest.json`을 수정하여 필요한 권한 추가

## 참고 사항

1. 확장 프로그램은 일반 웹페이지의 기록에만 액세스할 수 있으며 Chrome 내부 페이지에는 액세스할 수 없습니다
2. 기록 가져오기는 최대 10,000개의 레코드로 제한됩니다
3. 각 도메인은 최대 30개의 기록 페이지를 저장합니다
4. 웹사이트 아이콘은 여러 타사 서비스(Yandex, Google, DuckDuckGo)에서 가져옵니다
5. 내부 네트워크 IP의 아이콘은 fetch를 통해 가져와 dataURL로 변환되므로 로딩 속도에 영향을 줄 수 있습니다
6. 아이콘 로딩이 실패하면 첫 번째 문자를 기반으로 하는 다채로운 아이콘이 폴백으로 표시됩니다
